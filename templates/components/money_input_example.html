Elementos com data-brl-text são automaticamente formatados como moeda brasileira.

Como usar:

  • • Adicione data-money="brl" aos inputs de dinheiro
  • • Durante a digitação: o valor não é reformatado (não interfere no cursor)
  • • Ao focar: remove formatação para edição fácil
  • • Ao sair do campo (blur): formata como "R$ X.XXX,YY"
  • • No submit: converte automaticamente para "1234.56" (formato backend)
  • • Para textos estáticos: use data-brl-text
